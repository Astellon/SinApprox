# SinApprox

テーブル引き、パデ近似でsinの高速化を頑張った痕跡。uint32版の位相クラスも入ってるが妥協が多いので参考程度にしてください。
ただし実験には影響がない部分です。

## テスト

```
$ g++ -std=c++14 -O3 phase.cpp sin.cpp test.cpp
$ ./a.out
```
